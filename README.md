# Ready For Unsteady –  Content

This repository contains public resources from [Ready For Unsteady](https://readyforunsteady.com).  
It includes printable checklists, infographics, and structured templates that can be freely viewed, downloaded, and shared with attribution.

**Purpose:**  
To make practical, beginner-friendly preparedness materials open and reusable for anyone who wants to be better prepared.

---

## What This Repository Includes

| Folder | Description |
|---------|-------------|
| **/checklists** | Printable preparedness checklists, such as water storage, go-bag, and power outage plans. |
| **/infographics** | Visual guides designed for quick learning and sharing. |

Only materials intended for public distribution are included. Internal drafts and private resources are excluded.

---

## How To Use

- Download and print any checklist or infographic for personal or community preparedness.  
- Share visuals on websites, newsletters, or social media with a link back to [readyforunsteady.com](https://readyforunsteady.com).  
- Adapt or translate materials under the same Creative Commons license.  
- Do not sell or use these materials for commercial purposes.

## About Ready For Unsteady

Ready For Unsteady helps individuals, families, and renters prepare for emergencies without panic.  
The goal is to make preparedness approachable, practical, and empowering.  
Visit [readyforunsteady.com](https://readyforunsteady.com) for more guides and resources.
